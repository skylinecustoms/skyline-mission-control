# Skyline Mission Control Dashboard

Mobile-responsive dashboard for monitoring all Skyline Customs business automations.

## Features

- **Real-time Dashboard** with smart update intervals
  - Working hours (7 AM - 8 PM): Updates every 5 minutes  
  - Overnight (8 PM - 7 AM): Updates every 3 hours
  - All updates handled by Haiku model for cost efficiency
